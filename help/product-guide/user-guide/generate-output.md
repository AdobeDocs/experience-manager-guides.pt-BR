# Geração de saída {#id1645H8040HD}

O Adobe Experience Manager Guides tem recursos de publicação incorporados para gerar saídas em uma variedade de formatos padrão do setor. A versão atual do Experience Manager Guides permite gerar saída nos formatos mais usados: AEM Sites, PDF, HTML5, EPUB, JSON e saída personalizada por meio de DITA-OT, publicação nativa do PDF e FMPS.

Como editor, você pode gerar várias saídas com apenas alguns cliques. Você pode gerar saída para um mapa DITA inteiro ou publicar seletivamente apenas alguns tópicos atualizados. Você também pode usar o recurso de publicação de linha de base para publicar seletivamente uma versão específica do mapa ou tópico DITA.

>[!NOTE]
>
> O Experience Manager Guides também oferece suporte à geração de saída para arquivos de mapa e de tópico do LwDITA.

Como especialista em produção, o Experience Manager Guides também facilita a automatização do processo de publicação, criando o Painel de publicação e executando fluxos de trabalho pós-publicação. O Painel de publicação também fornece uma visualização unificada de todas as tarefas de publicação enfileiradas e em execução no sistema. Você pode optar por finalizar ou cancelar qualquer tarefa de publicação no Painel de publicação. Você pode criar e associar seus modelos de design personalizados para gerar saídas em um layout específico. Além disso, o Experience Manager Guides permite usar plug-ins DITA-OT personalizados para reutilizar o processo existente de geração do PDF.
