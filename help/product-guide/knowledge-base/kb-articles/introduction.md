# Base de conhecimento Experience Manager Guides

A Base de conhecimento contém artigos práticos orientados por cenários para o Experience Manager Guides que abordam configurações, personalizações e casos de uso comuns, além da documentação dos produtos principais.

| Categoria | Descrição |
|---|---|
| Criação | Dicas e técnicas para criar e reutilizar conteúdo no Editor, incluindo personalizações do editor, gerenciamento de metadados e validação de conteúdo. |
| Estrutura de extensão do Guides | Saiba como personalizar a interface do usuário do Experience Manager Guides, desde a introdução à estrutura de extensão até a criação de componentes, widgets e a revisão das personalizações do aplicativo. |
| Source de dados externos | Orientação sobre como integrar e configurar conectores de fonte de dados externos com o Experience Manager Guides. |
| Publicação | Práticas recomendadas para geração de saída, incluindo PDF nativo, DITA-OT, modelos do AEM Sites e configurações de publicação baseadas em microsserviços. |
| Fluxos de trabalhos | Configure e use workflows de pós-geração para automatizar etapas após a produção da saída. |
| Tradução | Instruções sobre como melhorar o desempenho da tradução e gerenciar qual conteúdo está incluído ou excluído da tradução. |
| Casos de uso | Saiba mais sobre os cenários do mundo real, como conteúdo condicional, trabalho com chaves e variáveis de conteúdo contextuais na saída publicada. |
