# Personalizar editor {#id181AI0003PN}

O AEM Guides vem com um poderoso editor que permite que seus autores criem e editem documentos DITA. É possível personalizar a barra de ferramentas do editor para adicionar ou remover qualquer recurso que você possa acessar da barra de ferramentas. Além disso, você pode configurar o salvamento automático de arquivos, gerando IDs para elementos à medida que são inseridos em seus documentos e muito mais.
